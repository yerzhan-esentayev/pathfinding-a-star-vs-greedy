# Pathfinding: A* vs Greedy Best-First Search

## Description
This project compares two pathfinding algorithms on grid-based mazes:
- A*
- Greedy Best-First Search

The implementation uses a Manhattan-distance heuristic and visualizes how the algorithms perform in terms of path quality and search behavior.

## Technologies
- Python
- NumPy
- Matplotlib

## Features
- Random grid maze generation
- A* search implementation
- Greedy Best-First Search implementation
- Manhattan-distance heuristic
- Path visualization
- Algorithm comparison
